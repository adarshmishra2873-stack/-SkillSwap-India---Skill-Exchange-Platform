# SkillSwap India 🇮🇳
Created by Adarsh Kumar Mishra.

SkillSwap India is a free platform where users can exchange their skills without spending money. 

## Features
- Secure Google Login (Powered by Firebase Authentication)
- Real-time Global Chat System (Powered by Firestore)
- Mobile numbers and private details remain hidden
- Custom Admin Panel

## Tech Stack
- HTML5, CSS3, JavaScript
- Google Firebase (Auth, Firestore, Analytics)
